# Adatvédelmi Szabályzat
## NKE Event Analyzer Chrome Bővítmény

Utolsó frissítés dátuma: 2025. február 10.

### 1. Bevezetés

Ez az adatvédelmi szabályzat leírja, hogyan gyűjti és kezeli az adatokat az NKE Event Analyzer Chrome bővítmény. Elkötelezettek vagyunk a felhasználók adatainak védelme és a transzparens működés mellett.

### 2. Adatgyűjtés és Felhasználás

#### 2.1 Gyűjtött Adatok
A bővítmény kizárólag a következő adatokat olvassa és dolgozza fel:
- Az E-rendezvény oldalon megjelenő nyilvános eseményadatok
- A felhasználó szűrési beállításai (kizárólag lokálisan, a böngészőben tárolva)

#### 2.2 Mit NEM gyűjtünk
- Személyes azonosításra alkalmas információkat
- Bejelentkezési adatokat
- Böngészési előzményeket
- Cookie-kat vagy más nyomkövető adatokat
- Más weboldalakon található információkat

### 3. Adatok Tárolása és Megosztása

- Minden feldolgozott adat kizárólag a felhasználó böngészőjében, lokálisan kerül tárolásra
- Nem továbbítunk adatokat külső szervereknek
- Nem készítünk biztonsági másolatot vagy archívumot a feldolgozott adatokról
- Az adatok csak az aktuális böngészési munkamenetben érhetők el

### 4. Szükséges Engedélyek

A bővítmény a következő Chrome engedélyeket használja:
- `activeTab`: Az aktuális E-rendezvény oldal tartalmának olvasásához
- `tabs`: Az E-rendezvény oldal azonosításához

### 5. Harmadik Felek

A bővítmény nem használ:
- Külső analitikai szolgáltatásokat
- Hirdetési platformokat
- Harmadik féltől származó szkripteket vagy szolgáltatásokat

### 6. Felhasználói Jogok

A felhasználónak joga van:
- A bővítmény bármikori eltávolításához
- A lokálisan tárolt adatok törléséhez a böngésző adatainak törlésével

### 7. Működési Környezet

A bővítmény kizárólag a következő domain-en működik:
- https://rendezveny.uni-nke.hu/

### 8. Változások az Adatvédelmi Szabályzatban

- Az adatvédelmi szabályzat változásairól a bővítmény frissítésekor értesítjük a felhasználókat
- A jelentős változásokról külön értesítést küldünk
- Az aktuális szabályzat mindig elérhető a bővítmény dokumentációjában

### 9. Kapcsolat

Ha kérdése vagy észrevétele van az adatvédelmi szabályzattal kapcsolatban, keressen minket:
- A GitHub repository Issues szekciójában
- Az NKE hivatalos csatornáin keresztül

### 10. Megfelelőség

A bővítmény megfelel:
- Az EU Általános Adatvédelmi Rendeletének (GDPR)
- A Chrome Web Store irányelveknek
- Az NKE adatvédelmi szabályzatának

Ez az adatvédelmi szabályzat utoljára 2025. február 10-én került frissítésre.

# Privacy Policy
## NKE Event Analyzer Chrome Extension

Last updated: February 10, 2025

### 1. Introduction

This privacy policy describes how the NKE Event Analyzer Chrome extension collects and handles data. We are committed to protecting user data and maintaining transparent operations.

### 2. Data Collection and Usage

#### 2.1 Data We Collect
The extension only reads and processes the following data:
- Public event data displayed on the E-rendezvény (E-event) page
- User filtering settings (stored locally in the browser only)

#### 2.2 What We DO NOT Collect
- Personally identifiable information
- Login credentials
- Browsing history
- Cookies or other tracking data
- Information from other websites

### 3. Data Storage and Sharing

- All processed data is stored locally in the user's browser only
- We do not transmit data to external servers
- We do not create backups or archives of processed data
- Data is only accessible during the current browsing session

### 4. Required Permissions

The extension uses the following Chrome permissions:
- `activeTab`: To read content from the current E-rendezvény page
- `tabs`: To identify the E-rendezvény page

### 5. Third Parties

The extension does not use:
- External analytics services
- Advertising platforms
- Third-party scripts or services

### 6. User Rights

Users have the right to:
- Remove the extension at any time
- Delete locally stored data by clearing browser data

### 7. Operational Environment

The extension operates exclusively on the following domain:
- https://rendezveny.uni-nke.hu/

### 8. Changes to Privacy Policy

- Users will be notified of privacy policy changes during extension updates
- Significant changes will be announced separately
- The current policy is always available in the extension documentation

### 9. Contact

If you have questions or concerns about the privacy policy, please contact us:
- Through the Issues section of the GitHub repository
- Through official NKE channels

### 10. Compliance

The extension complies with:
- EU General Data Protection Regulation (GDPR)
- Chrome Web Store guidelines
- NKE privacy policy requirements

This privacy policy was last updated on February 10, 2025.
